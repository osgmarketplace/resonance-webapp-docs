A Collection is a group of products, what may often be referred to as a "Category Page" of some variety.
In reality, a collection is far more flexible and can be used to represent any
product grouping, and the collectionType can help differentiate between them.


```graphql
type PublicCollection {
  collectionId: ID!
  collectionType: String!
  content: CollectionContent
  demandHqId: ID!
  description: String
  featuredProductIds: [ID]
  primaryImageUrl: String
  products(
  after: String
  before: String
  first: Int
  last: Int
  manufacturer: [StringFilterInput]
  msrp: FloatRangeFilterInput
  price: FloatRangeFilterInput
  query: String
  slugs: [String]
  sortOrder: [ProductSortInput]
  standardizedColor: [ProductStandardizedColorFilterInput]
  tags: [StringFilterInput]
  vendor: StringFilterInput
): PublicProductConnectionWithSearch!
  publishedAt: Date
  seoDescription: String
  seoTitle: String
  slug: String!
  subtitle: String
  tags: CollectionTags
  title: String!
}
```


### Fields

#### `collectionId` ([`ID`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/id))




#### `collectionType` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Any string to declare a category of this collection, can be used to filter collections, limit 128 characters.


#### `content` ([`CollectionContent`](/developers/demandhq-apis/storefront-api/graphql/reference/objects/collection-content))

Get the content of the collection.


#### `demandHqId` ([`ID`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/id))




#### `description` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

A short description of the collection, limit 1000 characters.


#### `featuredProductIds` ([`ID`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/id))

This can be used to pin certain products to the top of a collection when using certain sorts.
Max number of products is 20.


#### `primaryImageUrl` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

A primary image url for this collection, can be generated from a Collection ContentItem.


#### `products` ([`PublicProductConnectionWithSearch`](/developers/demandhq-apis/storefront-api/graphql/reference/objects/public-product-connection-with-search))

Paginate through products in a collection.

Requires the publicproduct/read scope.
- ##### `after` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Returns the elements in the list that come after the specified cursor.


- ##### `before` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Returns the elements in the list that come before the specified cursor.


- ##### `first` ([`Int`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/int))

Returns the first n elements from the list.


- ##### `last` ([`Int`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/int))

Returns the last n elements from the list.


- ##### `manufacturer` ([`StringFilterInput`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/string-filter-input))

Optionally filter by manufacturer.


- ##### `msrp` ([`FloatRangeFilterInput`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/float-range-filter-input))

Optionally filter by msrp.


- ##### `price` ([`FloatRangeFilterInput`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/float-range-filter-input))

Optionally filter by price.


- ##### `query` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Query products for certain terms. Searches several fields, including
product.tags, product.title, product.vendor, productLocales.description,
productLocales.featureBullets, variant.barcode, variant.mpn, variant.sku


- ##### `slugs` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Filter Products using slugs


- ##### `sortOrder` ([`ProductSortInput`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/product-sort-input))

Sort Products by a series of fields and directions, defaults to CREATED_AT/ASC.


- ##### `standardizedColor` ([`ProductStandardizedColorFilterInput`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/product-standardized-color-filter-input))

Optionally filter by standardized color.


- ##### `tags` ([`StringFilterInput`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/string-filter-input))

Optionally filter by tags.


- ##### `vendor` ([`StringFilterInput`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/string-filter-input))

Optionally filter by vendor.


#### `publishedAt` ([`Date`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/date))

ISO format date when the collection was published. Note that ensuing updates
with status set to PUBLISHED will republish and re-set this date.


#### `seoDescription` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Description specific for SEO purposes, limit 170 characters.


#### `seoTitle` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Title specific for SEO purposes, limit 256 characters.


#### `slug` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Slug for the URL for this collection. Limit 128 characters.


#### `subtitle` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

The subtitle of the collection, limit 256 characters.


#### `tags` ([`CollectionTags`](/developers/demandhq-apis/storefront-api/graphql/reference/objects/collection-tags))

Get the tags of the collection.


#### `title` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

The title of the collection, limit 256 characters.
